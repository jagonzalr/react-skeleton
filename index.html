react-skeleton

A skeleton React app with Gulp and Skeleton.

Styles

  • Normalize CSS
  • Skeleton Responsive CSS Framework

Batteries

  • Babelify
  • Browserify
  • Browsersync
  • Inject of CSS and JS
  • Minification

Usage

  • Download the zip file
  • Install dependencies npm install
  • Run project npm start or gulp
  • To build for distribution / production gulp dist
